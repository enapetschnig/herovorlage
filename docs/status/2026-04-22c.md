## Was die App jetzt kann (End-to-End-Demo)
1. Login mit `admin@demo.heatflow.local` / `demo1234`
2. Dashboard sehen, KPIs/Pipeline/Activity-Feed
3. Cmd+K drücken → „Steiner" tippen → mit Enter zum Kontakt/Projekt/Dokument springen
4. Auf `/documents` Liste durchgehen, AN-2026-001 öffnen → A4-Vorschau mit allen 10 Positionen, Brutto €31.320
5. „Als Rechnung" klicken → automatisch RE-2026-001 mit gleichen Positionen erzeugt, Status `draft`
6. Zum Steiner-Projekt → Logbuch-Tab → Eintrag hinzufügen → erscheint sofort
7. Files-Tab → PDF/Bild per Drag&Drop hochladen → in Supabase Storage Bucket `project-files`
8. Dokument abschließen → wird `locked`, danach nur noch Kopie möglich (GoBD)
