## Done in dieser Runde

### Zeiterfassung (CLAUDE.md Teil I.6)
- **`time` tRPC-Router** (`packages/api/src/routers/time.ts`):
  - `byDay({ userId, date })` — alle Einträge eines Users für einen Tag, mit Joined-Project-Title und Category-Name; rollt `workMinutes`, `pauseMinutes`, `billableMinutes` auf
  - `byProject({ projectId })` — Aggregat pro User für ein Projekt (für Projekt-Detail-Tab)
  - `weeklySummary({ fromDate, toDate, userId? })` — Reporting-Endpunkt
  - `create / update / remove` — manuelle Einträge mit automatischer Dauer-Berechnung
  - `quickStart({ projectId, activityType })` — startet einen laufenden Eintrag, **stoppt automatisch alle anderen laufenden Einträge desselben Users**
  - `quickStop()` — stoppt den aktuell laufenden Eintrag, berechnet Dauer
  - `running()` — gibt den aktuell laufenden Eintrag des Users zurück (für Live-Timer-Anzeige)
  - `approve({ id })` — Foreman/Owner-Aktion, setzt `approved_by_user_id` + `approved_at`
  - `categories()` — listet `time_categories` des Tenants
- **Time-UI** (`/time?date=YYYY-MM-DD&user=...`):
  - **HERO-Style 2-Kacheln-Layout**: Arbeitszeit + Pausenzeit, jede mit aufsummierter Stundenanzeige in Großbuchstaben, „+ Eintrag" / „+ Pause" Button
  - **KPI-Bar oben**: Arbeitszeit, Pausenzeit, abrechenbar, Saldo (8h-Soll mit ±-Anzeige in success/warning Farbe)
  - **Quick-Timer-Card**: Wenn ein Timer läuft → grüner Puls + Live-Dauer + Stop-Button; sonst → Activity-Selector + Projekt-Dropdown + Start-Button
  - **Inline-Edit pro Eintrag**: Pencil-Icon klappt 4 Felder auf (Start/Ende/Kategorie/Projekt + Kommentar) — onBlur speichert direkt
  - Datums-Navigation mit ← → Buttons + „Heute"-Shortcut + Date-Picker
  - Admin/Owner kann andere Mitarbeiter via Dropdown wählen
- **Projekt-Detail: neuer Tab „Zeit"** zeigt Aggregat pro User mit Total-/Billable-Stunden, klickbar zur User-Tagesansicht

2. Sidebar **Zeit** → Tagesansicht 2026-04-21 → Thomas hat 7:30h Arbeitszeit, kein Saldo (8h-Soll: −0:30h, warning)
3. Quick-Timer: Aktivität „Fahrt" wählen, Steiner-Projekt selektieren, **Start** → grüner Puls oben → später **Stop**
4. Manueller Eintrag: + Eintrag → 8-stündiger Default mit 30 Min Pause → inline Pencil → Kategorie/Projekt/Kommentar nachpflegen
5. Projekt-Anlegen: in der Combobox „Stefan" tippen, mit Enter Quick-Dialog öffnen, Kontakt anlegen → ist sofort selektiert, Projekt kann gespeichert werden
6. Kontakt/Projekt **Bearbeiten** über Detail-Seite → vorbefülltes Formular → Speichern → Detail wird sofort refresht
